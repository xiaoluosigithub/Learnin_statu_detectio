### 安装步骤

1. 克隆或下载本项目到本地

2. 安装所需依赖：
   ```
   pip install -r requirements.txt
   ```

3. 准备必要的资源文件：
   - 创建`images`文件夹，在其中放入cover.png和icon.ico
   - 创建`model`文件夹，在其中放入shape_predictor_68_face_landmarks.dat

   > 注：shape_predictor_68_face_landmarks.dat模型文件可从dlib官方网站下载

### 使用方法

1. 运行主程序：
   ```
   python main.py
   ```
